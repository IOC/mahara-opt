Etiquetes

Podeu posar Etiquetes als artefactes i pàgines que creeu al portafolis. Les etiquetes us permeten afegir informació descriptiva a artefactes i pàgines així com crear un índex classificador d'etiquetes.

Per associar diferents Etiquetes a un artefacte o pàgina separeu cada Etiqueta amb una coma. Per exemple: 'Avaluació', 'Trimestre d'hivern', 'Tutoria'.

Si feu clic a Mostra les meves etiquetes veureu una llista de les Etiquetes que heu usa. Si feu clic sobre el nom d'una d'aquestes Etiquetes existents s'afegirà automàticament a la llista d'Etiquetes de l'artefacte o pàgina que esteu creant o editant.

Podreu trobar tots els artefactes i pàgines que heu associat a una determinada Etiqueta si la cerqueu a Cerca al Meu Portafoliscreant així un sistema d'arxiu o l'índex del portafolis.
